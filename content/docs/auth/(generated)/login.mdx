Este endpoint permite que os operadores se autentiquem fornecendo suas credenciais. Após a autenticação bem-sucedida, ele gera e retorna um token de acesso pessoal que pode ser usado para acessar recursos protegidos. O token é essencial para fazer solicitações autenticadas a outros endpoints dentro do sistema.
